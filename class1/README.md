## Git 사용방법

### Git이란?

버전 관리도구. 코딩을 할 때 강력한 협업도구

#### 버전관리는 왜 하는가?

* 코딩하다가 뭔가 잘못되었을 때 복구
* 변경사항 추적
* 협업을 할 때 누가 그 코드를 작성했는지 알기 위해서
* 협업을 할 때 코드를 잘 합치기 위해서

### 기본 개념
